# StudentManagementSystem (console-based java project)
This is a simple "Foundational Java" project built using basic core Java concepts.  
It runs entirely in the terminal and allows the user to manage student records easily.

## Features
* Add a new student  
* View all students  
* Search a student by ID  
* Delete a student by ID  
* Exit the application

## Technologies Used
* Java (Core Java)
* OOP Concepts
* Classes & Objects  
* ArrayList  
* Methods  
* Encapsulation

### **Main.java**
* Runs the program  
* Shows menu  
* Takes user input  

##  How to Run the Project

1. Install **Java (JDK 8 or above)**  
2. Open your terminal/command prompt  
3. Navigate to the project folder  
4. Compile the files:

javac Main.java

5. Run the project:

java Main
